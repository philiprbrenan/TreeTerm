# Create a parse tree from an array of terms representing an expression.

![Test](https://github.com/philiprbrenan/TreeTerm/workflows/Test/badge.svg)

```
ok test [qw(v1 a2 v3 d4 v4)], <<END;
    a2
 v1       d4
       v3    v4
END
```
